# @mfilin/eslint-config

Extendable configuration for eslint

## Installation

```bash
npm i --save-dev @mfilin/eslint-config
```

## Configuration

In package.json:

```json
"eslintConfig": {
  "extends": "@mfilin/eslint-config"
}
```
